# Personal Portfolio Website

- Modern and clean UI
- Fully responsive design
- Built with Next.js and React
- Smooth user experience
- Sections for About, Skills, Projects, and Contact
- Optimized performance

## 🛠 Tech Stack

- Next.js
- React.js
- Tailwind CSS
- JavaScript
